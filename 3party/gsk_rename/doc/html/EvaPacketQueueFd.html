eva_packet_queue_fd_new_by_family ()

EvaPacketQueue *    eva_packet_queue_fd_new_by_family   (int addr_family,
                                                         GError **error);

Create a new Packet Queue using a newly opened datagram socket of a given address family. The address family is the sequence of AF_ defines in the header &lt;sys/socket.h&gt; on most unices.

The address family of a EvaSocketAddress may be found using eva_socket_address_protocol_family().

addr_family :

 the system-specific address family.

error :

 optional pointer to an error to set if things go wrong.

Returns :

 the new packet-queue, or NULL if there is a problem creating the socket.

eva_packet_queue_fd_new_bound ()

EvaPacketQueue *    eva_packet_queue_fd_new_bound       (EvaSocketAddress *address,
                                                         GError **error);

Create a new Packet Queue using a newly opened datagram socket which is bound to a given address.

Note that socket address space for TCP and UDP is separate, so it's allowed (and sometimes encouraged) to bind to the same port for both a packet queue, and a stream-listener.

address :

 the address to bind to.

error :

 optional pointer to an error to set if things go wrong.

Returns :

 the new packet-queue, or NULL if there is a problem creating the socket or binding.
