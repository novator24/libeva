Trivial Main Functions.

Trivial Main Functions. — These are trivial interfaces to the main-loop.

Synopsis

int                 eva_main_run                        ();
void                eva_main_quit                       ();
void                eva_main_exit                       (int exit_status);

Description

These function can be used to run the main loop for a fixed duration, or to quit / exit the program.

Details

eva_main_run ()

int                 eva_main_run                        ();

Run the main loop until it terminates, returning the value which should be returned by main().

Returns :

 the exit code for this process.

eva_main_quit ()

void                eva_main_quit                       ();

Quit the program by stopping eva_main_run().

eva_main_exit ()

void                eva_main_exit                       (int exit_status);

Exit the program by stopping eva_main_run().

exit_status :

 desired exit-status code for this process.
