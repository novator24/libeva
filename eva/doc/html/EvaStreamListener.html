Description

Derived classes of EvaStreamListener provide new EvaStreams as connections are accepted. Users can receive notification of this, and usually attach the stream to a new protocol object to handle the connection.

EvaStreamListenerAcceptFunc ()

gboolean            (*EvaStreamListenerAcceptFunc)      (EvaStream *stream,
                                                         gpointer data,
                                                         GError **error);

A method for handling a new connection.

Because eva_stream_attach() automatically adds a reference to the stream, the usual pattern is to do a few eva_stream_attach()s and then g_object_unref() from within this callback.
