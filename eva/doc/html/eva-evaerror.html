Description

This code principally consists of an enum GskErrorCode which can hopefully cover a wide range of error conditions. It encapsulates it in a GType'd enum so that you can get the name and nickname as usual. There is also support for mapping errno codes to GskErrorCode (eva_error_code_from_errno())
